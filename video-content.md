### 00:31 - 01:16 超高胜率与踩坑
- 星空策略试盘中用 **50U** 极小资金实盘
- 系统的悬壁(入场)和入场简直是"王者级别"
- 无心插柳，具备恐怖的爆发力，几乎一天就 **50%的收益**

### 01:16 - 04:57 泰勒展开式平仓的设想
- 受之前现货逻辑影响，最初设计是 **SIGMOID进SIGMOID平**
- 但做空策略本质是**逃顶**，具备很强的**不对称性**
- SIGMOID退出机制的盈亏比会非常差
- 思路源头依然是**泰勒展开式(Taylor Expansion)**，回归做空策略的最初思路：**求导**和**降次**
- 入场逻辑：
  - 策略入场开启时认为此时是**顶部**
  - 因为一阶导数接近0，二阶导小于0
- 根据泰勒展开式，对未来一个delta t（如15分钟）的价格预测，本质是**函数寻根**过程
- **预测残差(Prediction Residual)** 概念引入：
  - 真实价格 > 预测价格 → 上移过大 → 猜想被证伪 → 立即止损
  - 预测与现实精准贴合 → 下跌稳健 → 等待下一个导数为0的点自动止盈
  - 真实价格暴跌远超预期 → 向下残差过大 → 预测价格点作为**移动止损(trailing stop)**
- 该想法无意中触碰了**现代控制理论(Modern Control Theory)**：**MPC**与**卡尔曼滤波(Kalman Filter)**中的残差反馈

### 04:57 - 05:58 平仓设想的致命缺陷
- 仓位管理全部使用SIGMOID，残差能否压入到同一个SIGMOID？
- 可以，但**绝不能线性叠加**，否则遗传算法(GA)会乱
- SIGMOID是单调递增/递减函数，但业务诉求是**双边**的：
  - 猜错了 → 平仓
  - 猜得太对 → 保持仓位
  - 猜得刚刚好 → 不需要处理，让自动指引来平仓
- 需要将残差转为**偏离惩罚(deviation penalty)**，压进线性层
- **致命漏洞**：
  - 做空策略GA进化出来的是 **KA和KV**（模式识别的Z空间，代表主观敏感度）
  - 但泰勒预测公式中根本没有KV和KA
  - 降次过程中已把**原方程抛弃**了
  - **"怎么可以用导函数去预测原函数？"**
  - 原函数求导变成速度V和加速度A时，**常数(绝对价格)已被抹除**
  - 开仓时GA只看V和A两个导数特征，不管绝对价格——找拐点只需要导数
  - 但**平仓阶段要预测未来价格，必须通过原函数计算**
  - 用导函数的V和A去预测 = **"用明朝的剑斩清朝的官"**

### 05:58 - 09:42 终极破局
- 必须彻底放弃SIGMOID的耦合，**完全脱离**开仓逻辑
- 引入一条全新的、与开仓彻底独立的管线
- 平仓是一个极其**干净的独立机制**，不能和其他逻辑混在一起
- 开仓那一瞬间**拍下物理快照**：
  - 冻结 **P0（价格）、V、A** 作为**原函数的初始边界条件**
  - 由此画出**预测泡沫线**
- **实战应用**（第一性原理）：
  - 在时间T足够短的微观尺度下抓住极值点
  - 局部走势 = 完美的**三次方程(cubic equation)**
  - 开仓点是顶点（一阶导为0）→ 下一个拐点必然是**最低点** = 精确平仓位置
- **卡壳点**：时间不是瞬时的，是连续的，一分一分往前跑
  - 随着K线跳动，怎么知道一开始预测的三次方程还准不准？
  - 一度想引入**军工级alpha-beta-gamma滤波器**（像预测火箭轨道一样每分钟修正系数）
  - 但发现问题：如果每分钟都在修正轨迹，目标低点会不断偏移
  - **等于承认开仓模型是错的 → 错为什么还要在这里开仓？**
- **正确解法**：
  - 把开仓那一瞬间解出来的三次函数死死钉住，作为**唯一真源**
  - 不管时间怎么走，**不需要重算系数**
  - 把每分钟真实价格与三次曲线方程上的对应值做对比
  - 通过**残差**计算，衡量现实世界与最初完美设想之间的**偏移度**
  - 真实价格紧贴黄线 → 拿住满仓直到吃掉终极低点
  - 真实价格偏移过大 → 模型猜错，原地止盈或止损
- 这就是真正的**解耦(decoupling)**：
  - 用最纯粹的原函数算出来的残差
  - 喂给专属平仓的第二个SIGMOID函数

### 09:42 - 10:00 导函数与原函数对齐！
（此段为过渡总结，强调开仓用导函数(GA找拐点)和平仓用原函数(泰勒预测价格)的分工对齐）

### 10:00 - 15:22 平仓逻辑五条铁律

**铁律一：利润的硬门槛**
- 扣动扳机那一刻，三次方程注定的低谷
- 利润空间必须覆盖**双边手续费**，否则不配成为目标

**铁律二：动态引力挂单**
- 下单瞬间就挂止盈单，目标死死盯在谷底
- 每分钟现实价格紧贴基准线（残差小）→ 平仓SIGMOID自信度趋近于1 → 挂单毫不退让，**榨干最后一滴利润**
- 价格开始偏移（残差变大）→ 自信度衰减 → 挂单被引力拉回现实，**主动向上保守退让**确保成交

**铁律三：绝不重算**（最关键的一条）
- 自信度跌破阈值 → 真实K线彻底脱离预测线
- **"不要重算系数去骗自己了，错了就是错了"**
- 直接**一键全屏止损**

**铁律四：All or Nothing**
- 在非常微观的战斗中，不要搞分批减仓、复盈加仓这种操作
- 一旦触发平仓条件，**一次性出清**，绝不拖泥带水

**铁律五：拔网线级别的止损**
- 开仓后偏移度瞬间向对立面暴躁拉伸（如迎头撞上大阴线）
- 说明开仓点彻底看错
- **废除前面一切优雅挂单逻辑，直接市价全屏赶紧跑**
- 为防止正常波动误判，做**平滑处理**：
  - 传入SIGMOID的残差不能是单分钟K线
  - 需要引入**EMA**式的平滑计算和**半衰期(half-life)**概念
  - 半衰期设定为 **15分钟**
  - 刚入场时效率最高（t小，衰减因子≈1），系统对极致的反转敏感
  - 度过15分钟考验期 → 衰减因子降低 → 三次项衰减 → 预测曲线变平缓
  - 高次项衰减为0 → 变成**二次函数跑步线** → 再转成**直线**
  - 每隔45分钟重新挂移动止盈

### 15:22 - 16:27 多币种做空策略的特殊GA特性
- 做空策略的GA回测思路特殊：
  - 这是一个**离散型事件驱动**策略，与绝对时间无关
  - 不能拿它跑全生命周期的完整回测——没有意义
  - 要用 **"拼K"(K-patch)** 概念：
    1. 引入**宏观门控(macro-gating)**
    2. 只把近期涨幅超过50%的标的在三周内的K线切片
    3. 或符合**泡沫密度律公式**(bubble density formula)预测器的K线事件提出来
    4. **拼到一起**，只对这些内核切片做进化回测
    5. 这样通过GA才能算出真实的解

### 16:20 - 16:27 结尾忠告
- "最后给大家一个忠告：**能不做合约就不要做了，太累了。** 拜拜。"

## 核心论点提炼

### 论点1：止盈止损是量化交易中最容易被忽视的"巨坑"
- 入场逻辑可以通过Vibe Coding + GA进化出优秀的表现（"进场猛如虎"）
- 但**平仓逻辑如果不独立设计，会导致整个系统崩溃**（"出场二百五"）
- 作者用50U实盘证明：即使入场胜率极高，平仓系统的崩溃也能让所有利润一夜归零

### 论点2：开仓和平仓必须完全解耦（Decoupling）
- **致命错误**：最初用SIGMOID同时处理开仓和平仓
- 做空策略的本质是逃顶，具有**强不对称性**，与SIGMOID的单调性冲突
- 解决方案：平仓是一条**完全独立的管线**，不能和开仓逻辑混合
- 开仓用GA在Z空间找拐点（只看导数V/A），平仓用原函数预测价格（需要绝对价格P0）

### 论点3：导函数不能预测原函数——"明朝的剑斩清朝的官"
- 降次过程中已将原方程抛弃（绝对价格被抹除）
- GA在Z空间中进化出的是KV和KA（速度/加速度敏感度）
- 但泰勒预测公式需要**绝对价格P0**——维度根本不匹配
- 必须在开仓瞬间冻结P0、V、A作为**物理快照/边界条件**

### 论点4：一旦开仓，绝不重算——相信初始模型
- 曾想引入alpha-beta-gamma滤波器每分钟修正系数（像跟踪火箭轨迹）
- 但发现修正=承认初始判断是错的→那为什么还要开仓？
- **正确做法**：开仓时刻的三次方程是"唯一真源"，不做重算
- 只计算真实价格与预测曲线之间的残差，衡量偏移度

### 论点5：平仓逻辑五条铁律形成完整闭环
1. **利润门槛**：必须覆盖双边手续费
2. **动态引力挂单**：残差小→吃掉全部利润；残差大→保守退让确保成交
3. **绝不重算**：错了就是错了，直接止损
4. **All or Nothing**：微观战斗不分批
5. **拔网线止损**：极端情况废掉一切，立即跑

### 论点6：做空策略的GA回测不能用全生命周期法
- 做空策略是**离散型事件驱动**（与时间无关，与事件有关）
- 全生命周期回测没有意义
- 应使用 **"拼K"(K-patch)** 方法：
  - 宏观门控筛选符合条件的K线切片
  - 拼接到一起做进化回测
  - 才能算出真实的GA解
